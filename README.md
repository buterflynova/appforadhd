# ReplyGuy 💬
> Don't ghost your favorite people.

**ReplyGuy** is a lightweight cross-platform mobile application built with React Native and Expo. It is designed specifically to combat "text fatigue" and help users keep track of when to text back their friends, family, and loved ones before too much time slips away.

---

## ✨ Features

*   **Custom Reminders:** Add your friends and set a personalized response interval (e.g., text back every 7 days).
*   **Live Countdown:** Dynamic status indicators show you exactly how many days you have left to reach out.
*   **Visual Urgency:** Cards turn dark red and flag an "Overdue" status when you've gone past your target window.
*   **One-Tap Reset:** Click the checkmark button when you finally text them to instantly reset their countdown timer to today.
*   **Dark Mode UI:** Sleek, modern slate-dark user interface that looks sharp and is easy on the eyes.

---

## 🛠️ Tech Stack

*   **Framework:** [React Native](https://reactnative.dev/) (Cross-platform iOS & Android development)
*   **Tooling:** [Expo](https://expo.dev/) (SDK 54)
*   **Icons:** [Lucide React Native](https://lucide.dev/)
*   **Language:** JavaScript (ES6+, React Hooks)
